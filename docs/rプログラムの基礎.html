3.2 いつRを使いたいか

要点 - Rの強みは、ユーザーベースが広いことと、再現可能性に優れている点である - Rの弱みは、計算効率があまり高くないことである

… プログラミングの基礎 - オブジェクトとは、オブジェクト名という「名札」のついた情報を入れられる「箱」である - オブジェクト名は、他人が読んで分からないほど省略しすぎない - コメントをせずとも、関数や変数の「名札」を使って、分析内容が分かるようにする
